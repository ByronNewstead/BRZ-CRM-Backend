# BRZ-CRM-Backend
BRZ-CRM is a proprietary CRM platform designed for scalability, combining a custom-built frontend with a Python backend. It delivers powerful dashboards, pipeline management, real-time updates, and a flexible design system to streamline workflows and boost productivity.

These files are specifically incharge of the NodeJS backend. Specifically the Server.js and the package.json which is incharge of importing all the packages the server and system needs.
